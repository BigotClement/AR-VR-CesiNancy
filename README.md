# AR-VR-CesiNancy

Ce jeu, créé pour un projet d'étude, est jouable sur plusieurs plateformes : 
  - PC
  - HTC Vive

Chaque plateforme représente une équipe, les joueurs ont la possibilité de tirer pour tuer leurs adversaires.
